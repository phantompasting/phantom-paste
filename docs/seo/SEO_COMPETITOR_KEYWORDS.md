## TL;DR — Use These Terms, Avoid Those

### Use (generic, safe, high volume)
- **wheat pasting** / **wheat paste** (two words — generic craft term, predates any TM)
- **poster campaigns** / **poster advertising** / **wheat paste poster campaigns**
- **street postering** / **street posters** / **street poster advertising**
- **flyposting** / **fly posting** (UK-origin, no US TM, wildposting.com itself calls it "generic")
- **guerrilla marketing** / **guerrilla advertising** / **guerrilla postering**
- **street-level advertising** / **street-level marketing**
- **snipe posters** / **snipe advertising** (format-specific, generic)
- **sidewalk stencils** / **chalk stencil activations** / **sidewalk decals**
- **pole wraps** / **pole wrap advertising**
- **OOH advertising** / **out-of-home advertising** / **alternative OOH**
- **urban art marketing** / **street-level brand activation**
- **wheatpaste murals** (generic descriptive)
- **poster campaign agency** / **wheat pasting agency**

### Avoid (registered / claimed trademarks)
| Phrase | Holder | Evidence |
|---|---|---|
| **Wild Posting®** | wildposting.com | ® symbol used throughout site; H1 is "WILD POSTING®" |
| **Wheatpaste Posters™** | Printing Digital USA Inc / wheatpasteposters.com | ™ symbol, "The Original Since 2002" tagline |
| **Wheatpasting™** | Same (Printing Digital USA Inc) | Claimed as trademark on wheatpasteposters.com |
| **Guerrilla Gorilla™** | gogorillamedia.com | Branded mark |

> ⚠️ **Important nuance:** Generic descriptive usage of "wheat pasting" (two words, as verb or craft term) is **defensible** — it predates any brand registration. What we avoid is using these phrases *as brand names* or in ways that could be read as trading on someone else's mark.

## Content Gaps vs Competitors (Ranked Opportunities)

1. **"Wheat pasting company" / "wheat pasting agency"** as a brand category — no competitor owns this exact phrase. Phantom Pasting can claim it. Add to hero, nav, layout meta.

2. **Field-note content voice** — No competitor publishes installer-written how-to with specific field detail (water sourcing, wall-type paste formulas, broom selection, night install logistics). This is the EEAT moat per the blog plan.

3. **Photo-documented proof** — Most competitors show aspirational renders. We ship 100% daylight next-day photos. Lead every case study with the documentation.

4. **City-specific post-city pages with neighborhood detail** — Competitors list cities; none go neighborhood-deep. LA should carry Melrose / Silver Lake / Fairfax / DTLA / Echo Park; NYC should carry LES / Williamsburg / SoHo / Chinatown. Blog plan already specifies this.

5. **Flyposting / international pivot** — wildposting.com's own site admits "flyposting" is the generic term. No US agency has built a content silo around **flyposting** as an English-market alternative. Low-effort wedge.

6. **Legal / compliance content** — Only wheatpasteposters.com has surface-level compliance Q&A. None have installer-experience-based legal content ("what actually happens when police see us"). Blog Post #11 targets this gap.

7. **Cost transparency** — Nobody publishes real pricing. Blog Post #7 ("Wheat Pasting Cost 2026") targets this gap.

## Trademark-Defense Stance (Internal Guidance)

1. **Never** use `Wild Posting®` as an unacknowledged term in our copy — if we must reference it (e.g., educational blog comparing category terms), attribute: *"a trademarked trade term used by other agencies"* without naming.

2. **Never** use `Wheatpasting™` as one word in a brand/tagline/H1 context. Two-word **"wheat pasting"** as a verb/gerund is generic and safe.

3. **Never** use `Wheatpaste Posters™` as a caps brand phrase. Lowercase generic "wheat paste posters" as a descriptor (e.g., "we install wheat paste posters") is generic and safe.

4. **Preferred framing** for our service: *"street postering agency"*, *"poster campaign agency"*, *"wheat paste campaign specialists"*, *"guerrilla postering company"* — all generic, none claimed.

5. If any of the above marks become a crawl-risk signal (e.g., copyright/TM notice from holder), immediately purge all references and replace with neutral descriptors.
