
## E-E-A-T Content Checklist (Every Case Study)

- [ ] Client name mentioned (with permission) or described specifically
- [ ] Campaign objective stated
- [ ] Specific neighborhoods/walls named
- [ ] Poster sizes and quantities mentioned
- [ ] Photo documentation embedded (2-3 hero images, 4-6 gallery)
- [ ] Result mentioned (even if qualitative: "immediate social pickup", "300+ organic IG reposts")
- [ ] Testimonial quote if available
- [ ] Related services linked
- [ ] Author byline: "By Phantom Pasting Crew"
- [ ] `Article` schema with `datePublished`
- [ ] `ImageObject` schema on all campaign photos
